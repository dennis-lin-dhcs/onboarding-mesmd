# React + TypeScript + MUI + HashRouter template

## Install

```bash
npm install
```

## Run locally

```bash
npm run dev
```

## Build static site

```bash
npm run build
```

## Preview the production build

```bash
npm run preview
```

## GitHub Pages note

If this will be deployed as a project site, update `base` in `vite.config.ts` to your repo name:

```ts
base: '/your-repo-name/'
```

Keep `HashRouter` in `src/main.tsx` for GitHub Pages.
